# Terraform Best Practices

## Overview

Terraform remains one of the most popular IaC tools in the industry whose functionality only keeps growing. Although it is impossible to follow a "one-size-fits-all" approach when it comes to building infrastructure, it should be feasible to define some general best practices that builders can work with. This project is a collection of some of the most important best practices to keep in mind when using Terraform.

---

## Structure

This project has been structured in the following way:

engineering principles

each engineering principle is divided into several subcategories

each subcategory is a collection of individual best practices
